# Frontend

- **Stack**: React functional components, hooks, Vite.
- **State**: Keep raw fetch/WS logic out of UI components. Prefer custom hooks like `useSession` and `useRelay`.
- **Persistence**: `sessionStorage` for active session recovery, `localStorage` for saved vaults.
- **Styling**: Tailwind CSS (dark-mode first).
- **Animations**: Framer Motion for transitions.
